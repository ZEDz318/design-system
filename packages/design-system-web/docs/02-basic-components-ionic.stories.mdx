# Basic Components (ionic)

Geovistory design system uses [ionic UI Components](https://ionicframework.com/docs/components) (buttons, cards, checkboxes, lists etc.) as basic building blocks to create more complex components. Ionic UI components are [open source](https://github.com/ionic-team/ionic-framework), shipped as [web components](https://www.webcomponents.org/introduction) (w3c standard), integrate perfectly with [Stencil](https://stenciljs.com/), are easy and fully [stylable](https://ionicframework.com/docs/theming/basics), are well [documented](https://ionicframework.com/docs/components), mature and look good! To see all available compontents click the button below:

<ion-button expand="block" target="_blank" href="https://ionicframework.com/docs/components">Official ionic components documentation</ion-button>
